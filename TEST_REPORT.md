## 📝 需要手动测试的功能

### 1. 用户注册和登录
1. 打开 http://localhost:3000
2. 点击"登录"或"注册"
3. 使用 Google 账号登录
4. 验证登录成功

### 2. 积分发放（测试用）
```bash
# 启动 Drizzle Studio
pnpm db:studio

# 在浏览器中打开 https://local.drizzle.studio
# 找到 credit 表
# 手动添加一条记录给测试用户
```

### 3. 视频生成流程
1. 登录后访问 http://localhost:3000/create
2. 上传照片和视频
3. 点击"生成"
4. 等待任务完成（约 15 分钟）
5. 查看结果

### 4. 支付流程
1. 点击"购买积分"
2. 选择套餐（Single 或 10-Pack）
3. 完成 Creem 支付
4. 验证积分到账
