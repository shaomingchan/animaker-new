# Archive Notes

This directory contains historical migration, handoff, and temporary project documents.

These files are kept only for reference.
They should not be treated as the primary source of truth for current development or production operations.

For current work, use the files in the repository root instead:

- `README.md`
- `AGENTS.md`
- `PRODUCTION_BASELINE.md`
- `PRODUCTION_CHECKLIST.md`

Reasons these files were archived:

- some documents conflict with current code
- some documents use older naming conventions
- some documents describe migration-era work rather than the live system
- some documents contain encoding corruption
